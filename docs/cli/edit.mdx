---
title: 'edit'
description: 'Edit the Suga application'
---

Edits an application using the suga.yaml application spec and referenced platform.

```bash
suga edit
```

## Features

- Visual infrastructure design interface
- Drag-and-drop components
- Real-time validation
- Export to code

## Example

```bash
suga edit
# Opening platform for live sync...
# Editor available at https://app.addsuga.com
```
