## suga new

Create a new Suga project from a template.

```bash
suga new [project-name]
```

**Options:**
- `--template` - Specify a template to use
- `--force` - Overwrite existing directory

**Examples:**
```bash
# Create project with default template
suga new my-app

# Use specific template
suga new my-app --template typescript-api

# Force overwrite existing directory
suga new my-app --force
```

## suga init

Initialize a new Suga project in the current directory.

```bash
suga init
```

Converts an existing directory into a Suga project.

## suga templates

List all available project templates.

```bash
suga templates
```

Shows available templates with descriptions.
