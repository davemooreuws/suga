## Quick Install

<Tabs>
  <Tab title="macOS">
    ```bash
    brew install nitrictech/tap/suga
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    scoop bucket add nitrictech https://github.com/nitrictech/scoop-bucket
    scoop install suga
    ```
  </Tab>
  <Tab title="Linux">
    ```bash
    curl -sSL https://suga.io/install.sh | sh
    ```
  </Tab>
</Tabs>

## Verify Installation

```bash
suga --version
```
