## Quick Install

<Tabs>
  <Tab title="macOS">
    ```bash
    brew install nitrictech/tap/suga
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    scoop bucket add nitrictech https://github.com/nitrictech/scoop-bucket
    scoop install suga
    ```
  </Tab>
  <Tab title="Linux">
    ```bash
    curl -sSL https://suga.io/install.sh | sh
    ```
  </Tab>
</Tabs>

## Verify Installation

```bash
suga --version
```

## Additional Requirements

**For local development:**

- [Docker](https://docs.docker.com/get-docker/) - Required for running services locally
- **Runtime (choose one):**
  - [Node.js 18\+](https://nodejs.org/) for TypeScript/JavaScript projects
  - [Python 3.9\+](https://www.python.org/downloads/) for Python projects
  - [Go 1.21\+](https://golang.org/dl/) for Go projects

**For cloud deployment (optional):**

- [Terraform CLI](https://terraform.io/downloads) - Required for infrastructure deployment
- **Cloud provider CLI (choose one):**
  - [AWS CLI](https://aws.amazon.com/cli/) for AWS deployments
  - [Google Cloud CLI](https://cloud.google.com/sdk/gcloud) for GCP deployments
  - [Azure CLI](https://docs.microsoft.com/en-us/cli/azure/) for Azure deployments
