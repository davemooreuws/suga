---
title: 'Authentication'
description: 'Manage authentication with the Suga platform'
---

## suga login

Authenticate with your Suga account.

```bash
suga login
```

Opens your browser to complete authentication flow.

## suga logout

Log out from your Suga account.

```bash
suga logout
```

## suga access-token

Get an access token for programmatic access.

```bash
suga access-token
```

Used for CI/CD pipelines and automated workflows.
