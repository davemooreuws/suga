---
title: 'generate'
description: 'Generate client libraries for your application'
---

Generate client libraries for your Suga application.

```bash
suga generate
```

## Options

- `--language` - Target language for client library
- `--output` - Output directory

## Examples

```bash
# Generate TypeScript client
suga generate --language typescript --output ./client

# Generate Python client  
suga generate --language python --output ./python-client
```
