List all available project templates.

```bash
suga templates
```

Shows available templates with descriptions to help you choose the right starting point for your project.

## Example

```bash
suga templates
# Available templates:
# - typescript-api: TypeScript REST API template
# - python-web: Python web application template  
# - react-app: React frontend application template
```
