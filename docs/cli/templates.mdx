List all available project templates.

```bash
suga templates
```

Shows available templates with descriptions to help you choose the right starting point for your project.

## Example

```bash
suga templates
# Available templates:
# - suga/go-standard
# - suga/python-django-pip
# - suga/python-fastapi-uv
# - suga/typescript-express
```
