## Import

```go
import "example/suga"
```

## Usage

```go
// Initialize client
app, err := suga.NewClient()
if err != nil {
    log.Fatal(err)
}

// Access your resources (names from your suga.yaml)
app.Files.Write("key", []byte("data"))
content, err := app.Files.Read("key")
```
